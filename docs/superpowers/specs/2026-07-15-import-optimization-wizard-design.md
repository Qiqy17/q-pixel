# Q像素导入优化向导设计

**日期：** 2026-07-15

## 目标

让用户把图片快速转换为更贴合、更易制作的拼豆图纸。所有自动优化都必须可预览、可撤销，并保留原图和上一个版本。

## 设计原则

1. 默认简单，复杂参数按需展开。
2. 原图永远保留，不被优化过程覆盖。
3. 优化先预览，确认后才写入当前图纸。
4. “还原度”优先于盲目减少颜色。
5. 任何一步都可以返回修改。

## 导入向导

### 第 1 步：图片识别

导入图片后进入向导，展示原图和初步像素化结果。

自动处理：

- 判断图片是否已经是规则像素画。
- 估算网格列数、单元格大小、网格偏移。
- 生成初步图纸。
- 提供三种导入预设：高保真、均衡、易制作。

交互：

- 原图与图纸并排预览。
- 可切换单屏、左右对比和叠加对比。
- 提供“直接使用当前结果”和“进入精细调整”。

### 第 2 步：像素精度

用户可以调整：

- 网格列数。
- 单元格大小。
- 网格水平偏移。
- 网格垂直偏移。
- 画布宽度和高度。

预览要求：

- 调整后立即刷新局部预览。
- 支持放大查看眼睛、高光、轮廓等区域。
- 提供撤销上一步和恢复自动识别结果。
- 记录当前校准参数，重新打开项目时可继续调整。

### 第 3 步：色号优化

用户可以设置最多色号数量，并选择优化策略：

- 保真优先：尽量保留原始颜色和细节。
- 均衡：适度减少杂色，同时保护轮廓和高光。
- 易制作：减少颜色和零散孤立格，方便实际拼制。

AI识别规则：

- 自动识别轮廓色。
- 自动识别高光色。
- 自动识别眼睛、五官和小面积关键色。
- 关键颜色默认锁定，但用户可以解除锁定。
- 色号合并只允许作用于未锁定颜色。

结果展示：

- 优化前后色号数量。
- 被改变的格子数量。
- 被合并的颜色。
- 高光、轮廓和关键区域是否受到影响。
- 支持一键恢复优化前结果。

### 第 4 步：预览确认

确认页展示：

- 原图。
- 最终图纸。
- 原图叠加层。
- 色号数量。
- 总拼豆数量。
- 修改格子数量。
- 可能丢失的高光或边缘提示。

操作：

- 返回任意步骤。
- 重新识别。
- 保存为当前版本。
- 保留上一个图纸版本，支持恢复。
- 取消时不修改当前图纸。

## 版本和数据保护

导入向导生成的每次结果都先放在临时编辑状态中。只有点击“应用到当前图纸”后才替换正式图纸。

保存时保留：

- 原始图片。
- 当前校准参数。
- 当前导入模式。
- 色号优化配置。
- 关键锁定颜色。
- 上一个已确认版本。

## 后续扩展

第一阶段只做静态图片导入优化，不加入动画、复杂图层混合或自动艺术重绘。后续可增加：

- 分区制作和行列导航。
- 材料清单与缺色提示。
- 镜像、旋转和对称绘制。
- 多板拼接和实际尺寸 PDF。
- AI生成图与导入向导直接衔接。
