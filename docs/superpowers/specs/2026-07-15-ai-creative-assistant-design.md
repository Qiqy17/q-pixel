# Q像素 AI 创作辅助设计说明

## 目标

把 AI 生图从“一次生成、直接导入”升级为可控的创作流程，帮助用户得到更贴合主题、结构更清晰、色号更适合制作的像素作品。

## 第一阶段范围

1. **AI像素草图导演**：根据主题、风格、尺寸、色号上限和对称要求，生成结构化提示词，并在生成前显示可编辑摘要。
2. **多方案生成与对比**：一次请求最多生成 3 个候选，支持并排预览、选择一个进入导入，不自动替换当前画布。
3. **AI像素结构修复**：对选中的候选执行本地结构检查，修复高置信度的断线、孤立格和左右对称缺口，并保留撤销入口。
4. **AI配色设计师**：提供原图、主题色、易制作三类配色方案，映射到当前 MARD 色板，并显示色号数量和变化摘要。

## 交互流程

`输入主题 -> AI创作摘要 -> 生成候选 -> 选择候选 -> 结构修复 -> 配色方案 -> 进入现有导入向导`

候选阶段只保存在内存中。用户明确选择后才调用现有 `applyAiImageDataUrl`，因此取消或关闭不会改变当前作品。

## 边界与安全

- 远程接口只负责生成图片，不负责直接修改当前图纸。
- 结构修复和配色映射默认使用本地算法，失败时保留原始候选。
- 不自动覆盖用户已经锁定的轮廓和高光色。
- 三方案请求失败时回退为单方案，不阻塞原有 AI 生图流程。

## 验收标准

- 主题摘要可以被用户编辑后再生成。
- 生成界面能区分候选图、当前选中图和当前画布。
- 结构修复前后可切换，且撤销不会清除原图。
- 配色方案至少显示色号数量、孤立格数量和预计制作难度。
- 现有导入校准、原图对比层、色号优化和项目保存流程保持可用。
